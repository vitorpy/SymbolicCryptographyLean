# VCVio2

This directory contains a minimally modified version of the [VCVio library](https://github.com/Verified-zkEVM/VCV-io).  
The modifications consist of:

1. Applying small fixes for compatibility with our versions of Lean and Mathlib.  
2. Removing unused parts of the original library (to facilitate the compatibility fixes).
